Tạo trang web với HTML

1. Giới thiệu
HTML (Hyper Text Markup Language) là hàng loạt các đoạn mã theo quy ước được thiết kế để tạo nên các trang web và được hiển thị bởi các trình duyệt web. HTML cho phép bạn kết hợp văn bản với hình ảnh, âm thanh, video và các liên kết đến các trang siêu văn bản khác. HTML không phải là ngôn ngữ lập trình mà là ngôn ngữ đánh dấu.
